    • 搭建Github+Hexo个人博客

    Github Page

    在github上新建repository，名称为{username}.github.io

    搭建步骤

    安装Node.js

    Node.js安装简单，上官网下载即可

    有关配置见Node.js配置

    安装Git

    Git安装教程~亲测可行

    安装Hexo

    Node.js中带有npm命令，可以用这个命令下载Hexo:

    (c)npm install -g hexo-cli

    验证hexo安装成功:

    hexo -v

    创建本地博客

    1. 创建文件夹 blog
    2. 鼠标右键 blog，选择 Git Bash Here
    3. 输入hexo init
    4. 输入(c)npm install 安装依赖包
    5. 输入hexo g生成网页
    6. 输入hexo s生成的网页就部署在了本地服务器上，可以通过浏览器输入http://localhost:4000查看

    部署到Github

    上一步骤只是创建了本地博客，部署到Github需要以下步骤：

    1. github repository 复制url

    2. 修改博客配置文件_config.yml
      找到#Deployment写入：

      deploy:
      type: git
      repository: https://github.com/{username}/{username}.github.io.git
      branch: master

    3. 回到Git Bash中，执行以下命令：

      npm install hexo-deployer-git –save

    4. 输入hexo d，若有INFO Deploy done: git，则表示成功部署

    使用主题

    使用主题的思路就是下载主题文件复制到主题文件夹中，并在配置文件中配置使用的主题。操作步骤：

    1. 复制主题的github地址在Git Bash中输入:

      git clone {主题github url} themes/{主题名}

    2. 修改博客配置

      theme: 主题名
      language: zh-CN
      url: https://{username}.github.io

    3. 重新部署，在Git Bash中输入:

      hexo g -d

    可以通过在浏览器中输入{username}.github.io查看自己的博客
